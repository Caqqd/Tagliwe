# TagliWe - Create Your Dream Event

TagliWe is a modern, responsive event planning website.  
Designed for weddings, birthdays, and other events, it allows users to view services, packages, galleries, submit bookings, and contact the team directly via WhatsApp.

---

## Features
- Home page with hero banner
- Services page with cards
- Packages page with details & prices
- Gallery page
- Reviews page
- Booking form (stores in MongoDB)
- Contact page with WhatsApp & call buttons
- Mobile responsive UI

---

## Tech Stack
- **Backend:** Node.js, Express.js, MongoDB, Mongoose
- **Frontend:** React.js, Tailwind CSS
- **Notifications:** WhatsApp link
- **Optional:** Admin dashboard
